# Bounties

This repo manages bounties for various tasks. Each bounty type has its own folder with instructions and deliverables.

## Bounty types

- **[video-segmentation](video-segmentation/)** — Segmentation map plus edited master (intro/outro, subtitles, cuts)

*(More types will be added here.)*

## How it works

1. Open the folder for the bounty type you want.
2. Follow the instructions and submit a **Pull Request** with your deliverable.
3. Submissions go in that bounty type’s **submissions** folder (e.g. `video-segmentation/submissions/`). See each bounty README for the exact path and format.
4. **First accepted PR** per bounty gets paid after review.
