# Submissions

## Path

```
video-segmentation/submissions/<course-slug>/<language>/video-XX-YYYY_MM_DD.md
```


Example: `bitcoin-core-principles/en/video-01-2025_10_15.md`

## File contents

1. First line: `edited_video: <URL>` (public Drive link to the edited master).
2. **Segmentation table** only (strict format in parent README).

Do not commit video binaries; only the `.md` with link(s) to Drive.

First merged, accepted PR per video row wins the bounty.
